# Bài Tập JavaCore

Dự án này chứa các bài tập thực hành JavaCore, được tổ chức theo từng chủ đề và mức độ khó khác nhau.  
Mục tiêu là giúp ôn luyện và củng cố các kiến thức nền tảng về Java.

## 📂 Cấu trúc thư mục
- **src/** – Chứa mã nguồn Java của từng bài tập.
- **bin/** – (Nếu có) Thư mục chứa file `.class` đã biên dịch.
- **README.md** – Tài liệu hướng dẫn (file này).

## 📝 Nội dung chính
Một số chủ đề được triển khai trong các bài tập:
- Biến, kiểu dữ liệu và toán tử
- Cấu trúc điều kiện (`if`, `switch`)
- Vòng lặp (`for`, `while`, `do-while`)
- Mảng và chuỗi
- Hàm (method) và phạm vi biến
- Lập trình hướng đối tượng (OOP)
  - Lớp và đối tượng
  - Tính kế thừa
  - Tính đa hình
  - Tính đóng gói
- Xử lý ngoại lệ (Exception Handling)
- Làm việc với `ArrayList`, `HashMap` và các collection khác
- Đọc/ghi file trong Java

## 💻 Yêu cầu môi trường
- **Java JDK** 8 trở lên
- **IDE**: IntelliJ IDEA, Eclipse, hoặc NetBeans (khuyến nghị IntelliJ IDEA)
